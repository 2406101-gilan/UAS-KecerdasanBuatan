# UAS Kecerdasan Buatan

## Prediksi Status Gizi Balita Menggunakan KNN & Random Forest

Repository ini berisi tugas UAS mata kuliah Kecerdasan Buatan mengenai klasifikasi status gizi balita berdasarkan:

- Umur
- Jenis Kelamin
- Tinggi Badan

## Cara Menjalankan

1. Buka `uas_model.ipynb`
2. Upload `data_balita.csv`
3. Jalankan semua cell
4. Model akan tersimpan otomatis
